## Import Swagger ke Postman

1. Buka Postman  
2. Klik **Import**  
3. Pilih tab **Raw Text**  
4. Copy dan paste seluruh isi `openapi.yaml`  
5. Klik **Continue â†’ Import**

Postman akan otomatis membuat **collection lengkap** dengan semua endpoint.
